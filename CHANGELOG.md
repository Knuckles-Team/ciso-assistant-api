## [0.1.0] - 2026-06-15

### Added
- Initial release: CISO Assistant API client + MCP server + A2A agent.
- 100% coverage of the CISO Assistant REST API — a single vendored
  drf-spectacular spec, ~1,565 operations across 19 domains — generated by
  `scripts/generate_from_openapi.py`.
- 19 action-routed MCP tools (one per domain, mirroring the published
  documentation categories) plus a `custom_api` escape hatch.
- Knox token authentication (pre-minted token or username/password login),
  OIDC delegation, DRF `next`-link pagination, and `429`/transient-error
  retry handling.
- Coverage keystone test asserting spec operations == client methods == MCP actions.
